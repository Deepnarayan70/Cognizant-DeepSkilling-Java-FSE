# Data Structures & Algorithms (DSA)

## Overview

This folder contains the Data Structures & Algorithms (DSA) hands-on exercises completed as part of **Week 1** of the Cognizant Digital Nurture 5.0 (Java FSE) program.

The exercises focus on developing problem-solving skills, understanding algorithmic thinking, and implementing efficient solutions using Java. They cover searching techniques, recursion, and real-world programming scenarios.

---

## Exercises Completed

### Exercise 1 – Inventory Management System
Developed a simple inventory management application using Java classes and objects to perform basic product management operations.

### Exercise 2 – Search Algorithms
Implemented searching techniques to efficiently locate data within a collection and compared their working principles.

### Exercise 7 – Financial Forecast
Implemented a recursive solution to forecast future financial values based on historical data, demonstrating the practical application of recursion.
