# PL/SQL

## Overview

This folder contains the PL/SQL hands-on exercises completed as part of **Week 1** of the Cognizant Digital Nurture 5.0 (Java FSE) program.

The exercises focus on learning the fundamentals of **PL/SQL (Procedural Language/Structured Query Language)**, Oracle's procedural extension to SQL. They demonstrate how to write PL/SQL blocks, use variables and control structures, and develop reusable database programs.

---

## Exercises Completed

### Exercise 1 – PL/SQL Basics

Implemented basic PL/SQL programs to understand the syntax, structure, and execution of PL/SQL blocks.

### Exercise 2 – Control Structures

Worked with conditional statements and loops to perform logical operations within PL/SQL programs.

### Exercise 3 – Stored Procedures and Functions

Created reusable stored procedures and functions to perform database operations efficiently.
