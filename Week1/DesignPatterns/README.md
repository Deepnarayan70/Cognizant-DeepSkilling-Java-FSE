# Design Patterns

## Overview

This folder contains the Design Pattern hands-on exercises completed as part of **Week 1** of the Cognizant Digital Nurture 5.0 (Java FSE) program.

The exercises introduce commonly used **Creational Design Patterns** that help improve software design by promoting code reusability, flexibility, and maintainability. These implementations demonstrate how design patterns solve common software development problems using object-oriented programming principles.

---

## Exercises Completed

### Exercise 1 – Singleton Pattern

Implemented the Singleton Design Pattern to ensure that only one instance of a class is created throughout the application's lifecycle while providing a global access point to that instance.

### Exercise 2 – Factory Method Pattern

Implemented the Factory Method Design Pattern to create objects without exposing the object creation logic to the client. This promotes loose coupling and makes the application easier to extend and maintain.
