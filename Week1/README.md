# Week 1 – Java Fundamentals, DSA & Software Development Essentials

## Overview

Week 1 focuses on strengthening the core concepts of Java programming through Data Structures & Algorithms (DSA), Design Patterns, Unit Testing, Logging, Mocking Frameworks, and PL/SQL. The hands-ons provide practical experience in writing clean, maintainable, and testable Java applications while following industry-standard development practices.

Throughout this week, I implemented algorithms, explored software design principles, performed unit testing, worked with logging frameworks, and gained basic database programming experience using PL/SQL.
