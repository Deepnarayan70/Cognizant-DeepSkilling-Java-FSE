# JUnit

## Overview

This folder contains the JUnit hands-on exercises completed as part of **Week 1** of the Cognizant Digital Nurture 5.0 (Java FSE) program.

The exercises focus on learning the fundamentals of **unit testing** in Java using the JUnit framework. They demonstrate how to create test cases, verify expected results using assertions, and organize tests following industry best practices.

---

## Exercises Completed

### Exercise 1 – Setting Up JUnit

Configured a Java project with the JUnit framework and created the first unit test to verify application functionality.

### Exercise 2 – Assertions

Implemented various JUnit assertion methods to validate expected outputs and ensure application correctness.

### Exercise 3 – AAA Pattern and Test Fixtures

Applied the **Arrange-Act-Assert (AAA)** testing pattern and used setup and teardown methods to improve test organization and maintainability.
