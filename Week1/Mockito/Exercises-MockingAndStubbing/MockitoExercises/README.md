# Mockito

## Overview

This folder contains the Mockito hands-on exercises completed as part of **Week 1** of the Cognizant Digital Nurture 5.0 (Java FSE) program.

The exercises focus on learning **Mockito**, a popular Java mocking framework used for unit testing. Mockito enables developers to create mock objects, stub method behavior, and verify interactions between components without depending on actual implementations.

---

## Exercises Completed

### Exercise 1 – Mocking and Stubbing

Implemented unit tests using Mockito by creating mock objects, defining method behavior through stubbing, and verifying method invocations. The exercise demonstrates how Mockito simplifies testing by isolating business logic from its dependencies.
