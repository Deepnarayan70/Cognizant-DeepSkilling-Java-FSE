# SLF4J Logging

## Overview

This folder contains the SLF4J (Simple Logging Facade for Java) hands-on exercise completed as part of **Week 1** of the Cognizant Digital Nurture 5.0 (Java FSE) program.

The exercise focuses on implementing application logging using the **SLF4J** logging framework. Logging plays an important role in monitoring application behavior, debugging issues, and tracking execution flow during software development.

---

## Exercise Completed

### Exercise 1 – Logging with SLF4J

Configured the SLF4J logging framework and implemented logging statements to record application events, execution flow, and important runtime information. The exercise demonstrates how logging can be used instead of standard console output for better application monitoring.
