# Week 2 – Spring Core & Spring Data JPA

## Overview

This week focuses on learning the fundamentals of the **Spring Framework**, **Maven**, and **Spring Data JPA with Hibernate**. The hands-ons cover project creation, dependency injection, Maven project management, and database operations using JPA and Hibernate.

Through these exercises, I gained practical experience in building Spring applications, managing dependencies, and performing database operations using Spring Data JPA.

---

## Hands-ons Completed

### Spring Core & Maven

- Created a basic Spring application.
- Configured a Maven project.
- Implemented Dependency Injection (DI).
- Developed a simple Library Management application using Spring Framework.

### Spring Data JPA with Hibernate

- Created a Spring Boot project with Spring Data JPA.
- Configured database connectivity.
- Performed CRUD operations using JPA repositories.
- Explored the relationship between JPA and Hibernate.
- Implemented entity mapping and repository interfaces.
