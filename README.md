# Cognizant Deep Skilling – Java FSE

This repository contains my solutions, hands-on exercises, and practice work completed as part of the **Cognizant Deep Skilling Program (Java Full Stack Engineer)**.

The program covers core programming concepts, object-oriented programming, data structures, design patterns, Git & GitHub, React, Angular, DevOps, Docker, Cloud, and Generative AI. Each week's work is organized into separate folders for easy navigation.
