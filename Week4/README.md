# Week 4 – Microservices with Spring Boot

## Overview

This week focuses on developing applications using the **Microservices Architecture** with **Spring Boot** and **Spring Cloud**. The hands-ons demonstrate how multiple independent services communicate through an API Gateway while maintaining a modular and scalable architecture.

The exercises provide practical experience in creating, configuring, and managing multiple Spring Boot microservices.

- Loan Microservice
- API Gateway Configuration

## Projects

### Account Service
A Spring Boot microservice responsible for handling account-related operations.

### Loan Service
A Spring Boot microservice that manages loan-related functionalities and exposes REST endpoints.

### API Gateway
A centralized gateway used to route client requests to the appropriate microservices, providing a single entry point for the application.
